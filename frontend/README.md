# DocAI Frontend

React + Vite + Tailwind v4 frontend for the Agentic Document Intelligence pipeline.

See the [root README](../README.md) for full setup and usage instructions.

## Dev Commands

```bash
npm install       # Install dependencies
npm run dev       # Start dev server on :5173 (proxies API to :8000)
npm run build     # Production build to dist/
npm run preview   # Preview production build
npm run lint      # ESLint
```
